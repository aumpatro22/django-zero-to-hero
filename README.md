## 🛠️ Quick Local Setup

1. **Navigate to Django directory:**
   ```bash
   cd djnago
   ```
2. **Activate the Virtual Environment:**
   ```bash
   venv\Scripts\activate
   ```
3. **Run the Server:**
   ```bash
   python manage.py runserver
   ```
4. Open your browser to: [http://127.0.0.1:8000/products/](http://127.0.0.1:8000/products/)
